# ABX Crowdsale

## Local Development

1. Install MetaMask
2. Install and start Ganache
3. npm i
4. truffle compile
5. truffle migrate
6. npm run dev
7. Connect MetaMask to Ganache as a custem network
8. Use the Ganache recovery nmonic to get balaces
